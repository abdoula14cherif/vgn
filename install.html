Guide complet pour installer Kali Linux sur votre machine ou en machine virtuelle

Important - Sauvegardez vos données !

L'installation de Kali Linux peut effacer toutes les données de votre disque dur. Faites une sauvegarde complète de vos fichiers importants avant de commencer. Pour les débutants, nous recommandons d'utiliser une machine virtuelle.

Guide d'Installation Étape par Étape

1 Préparation du Média d'Installation

Téléchargez l'image ISO de Kali Linux depuis le site officiel et créez un média d'installation (clé USB ou DVD) :

# Pour créer une clé USB bootable sous Linux
sudo dd if=kali-linux-2023.3-amd64.iso of=/dev/sdX bs=4M status=progress

# Sous Windows, utilisez Rufus ou Etcher
# Sélectionnez l'ISO et votre clé USB, puis cliquez sur "Start"

Astuce

Utilisez Rufus en mode "DD Image" si vous avez des problèmes de boot.

2 Démarrage sur le Média d'Installation

Redémarrez votre ordinateur et accédez au menu boot (généralement F12, F2, ESC ou DEL). Sélectionnez votre clé USB ou DVD comme périphérique de démarrage.

Configuration UEFI/BIOS

Pour les systèmes modernes : Désactivez le Secure Boot dans les paramètres UEFI et activez le mode Legacy/CSM si nécessaire.

3 Menu de Démarrage de Kali

Après le démarrage, vous verrez le menu de Kali. Options disponibles :

Kali Linux Live (amd64) - Mode live sans installation
Kali Linux Installer - Installation graphique
Advanced options - Options de dépannage
Install with speech synthesis - Avec synthèse vocale

Sélectionnez "Graphical Install" pour l'installation standard.

4 Configuration de l'Installation

Suivez l'assistant d'installation étape par étape :

  • Langue : Sélectionnez votre langue (Français)
  • Localisation : Pays et fuseau horaire
  • Clavier : Disposition du clavier (Français)
  • Réseau : Nom d'hôte (ex: kali) et domaine
  • Utilisateur : Nom complet, nom d'utilisateur et mot de passe
  • Partitionnement : Option la plus importante

5 Partitionnement du Disque

Option recommandée pour débutants : "Guided - use entire disk"

Options de partitionnement disponibles :
1. Guided - use entire disk (Recommandé)
2. Guided - use entire disk with LVM
3. Manual (Expert)

Si vous avez Windows déjà installé, choisissez "Guided - use the largest continuous free space".

6 Installation des Paquets

L'installateur télécharge et installe les paquets de base. Assurez-vous d'avoir une connexion Internet active. Vous pouvez aussi utiliser le miroir HTTP.

Paquets Recommandés

Cochez "Graphical desktop environment" pour installer l'interface graphique. Pour une installation minimale, décochez toutes les options.

7 Installation du Chargeur de Démarrage (GRUB)

Installez GRUB sur le disque principal (/dev/sda généralement). Cochez "Yes" pour installer GRUB sur le MBR (Master Boot Record).

Installation de GRUB sur le périphérique :
/dev/sda (ou /dev/nvme0n1 pour les SSD NVMe)

Important : Ne pas installer GRUB si vous utilisez une VM !

8 Redémarrage et Première Connexion

Après l'installation, retirez le média d'installation et redémarrez. Connectez-vous avec le nom d'utilisateur et mot de passe que vous avez configurés.

Installation Terminée

Félicitations ! Kali Linux est maintenant installé. Exécutez sudo apt update && sudo apt upgrade pour mettre à jour votre système.

Dépannage et Problèmes Courants

Problème : L'écran reste noir après l'installation

Solution : Au démarrage, appuyez sur E dans le menu GRUB, ajoutez nomodeset à la fin de la ligne linux, puis F10 pour démarrer.

Problème : Carte WiFi non détectée

Solution : Connectez-vous en Ethernet et exécutez sudo apt update && sudo apt install firmware-atheros (adaptez au fabricant).

Problème : Dual Boot avec Windows ne fonctionne pas

Solution : Désactivez le Fast Boot dans Windows, puis utilisez sudo update-grub dans Kali.

Problème : Écran de connexion en boucle

Solution : Appuyez Ctrl+Alt+F2 pour passer en mode console, connectez-vous, puis réinstallez l'environnement de bureau : sudo apt install --reinstall kali-desktop-xfce
