Guide de Téléchargement

1

Choisir la Version Appropriée

Rendez-vous sur le site officiel kali.org/get-kali via les liens ci-dessus. Sélectionnez la version correspondant à votre matériel et besoins. Pour la plupart des utilisateurs, l'image ISO standard 64-bit est recommandée.

2

Téléchargement et Vérification

Cliquez sur le bouton de téléchargement. Le fichier ISO pèse environ 3-4 GB. Après téléchargement, vérifiez l'intégrité avec les sommes de contrôle SHA256 fournies sur le site.

3

Créer un Média d'Installation

Utilisez un logiciel comme Rufus (Windows), BalenaEtcher (multi-plateforme) ou dd (Linux) pour graver l'ISO sur une clé USB ou un DVD.

4

Démarrer sur le Média

Redémarrez votre ordinateur et configurez le BIOS/UEFI pour démarrer depuis la clé USB ou le DVD. Sélectionnez "Live Boot" pour tester sans installer, ou "Graphical Install" pour l'installation complète.

Vérification de l'Intégrité

Il est crucial de vérifier que votre téléchargement n'a pas été corrompu ou modifié. Utilisez les sommes de contrôle SHA256 fournies sur kali.org :

# Sous Linux/Mac
sha256sum kali-linux-2023.3-installer-amd64.iso

# Sous Windows (PowerShell)
Get-FileHash kali-linux-2023.3-installer-amd64.iso -Algorithm SHA256

Comparez le résultat avec la somme de contrôle affichée sur le site officiel. Si elles correspondent, votre téléchargement est intègre et sécurisé.
